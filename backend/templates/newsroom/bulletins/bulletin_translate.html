Translation Information

Creating a translation will:

  • Create a new bulletin linked to this original bulletin
  • Copy the intro and outro text for you to translate
  • Include the same stories in the same order as the original
  • Maintain the same categories as the original
  • Set you as the editor of the translation

After creating the translation, you will be directed to the edit page where you can translate the content.
